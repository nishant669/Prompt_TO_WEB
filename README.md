## 🎯 The Vision
**Prompt TO WEB** isn't just a website; it's a demonstration of **Zero-Code Architecture**. Built for the **2026 College AI Challenge**, this project explores how natural language prompts can be synthesized into production-ready front-end assets without traditional manual boilerplate.

> "The future of development isn't writing code—it's engineering the intent."

| Category | Stack | Implementation |

## 🛠️ Key Components & Innovation

<details open>
<summary><b>🖥️ Intelligent Interface (index.html)</b></summary>
<br>
The landing hub features a high-contrast dark theme. We used <b>AI-driven color theory</b> to select a palette that reduces eye strain while maintaining a premium "Cyberpunk" aesthetic.
</details>

<details>
<summary><b>📝 State-Driven Task Manager (new_task.html)</b></summary>
<br>
Unlike static pages, this module uses JavaScript to handle dynamic inputs. It demonstrates how AI can generate efficient event listeners and DOM manipulation logic.
</details>

<details>
<summary><b>🤖 The Prompting Strategy</b></summary>
<br>
We utilized <b>Recursive Prompting</b>:
1. **Definition:** Establishing global CSS variables.
2. **Iteration:** Generating components (Buttons, Cards, Modals).
3. **Refinement:** Passing code back to AI for minification and bug-fixing.
</details>
